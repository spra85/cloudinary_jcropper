cloudinary_jcropper
===================

JCrop photo cropper geared towards cloudinary transformations

Getting started
===========

`bower install`

Testing
===========

```
rake jasmine
```

Open a browser and visit `localhost:8888`

Via command line: 

```
rake jasmine:ci
```
