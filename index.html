Cloudinary jcropper

JCrop photo cropper geared towards cloudinary transformations

Download this project as a .zip file Download this project as a tar.gz file

Cropper options

  • debug - If true, will log crop coordinates as they change. (true|false, Default: false)
  • crops - Hash containing aspect ratio and width of each configured crop type. Default: (primary: { ratio: 4 / 3, width: 600 }, thumbnail: { ratio: 1, width: 60 })
  • originalPhotoSelector - Selector used to configure original photo to crop. Default: ".photo-cropper-original-photo"
  • cropOptionSelector - Selector used to configure each available crop type. Default: ".crop-selection"
  • saveCropSelector - Selector used to configure save button for crop(s). Default: ".save-crop"
  • cropInputSelector - Selector used to configure input field to store crop coordinates for each configured type. Default: "#crops"
